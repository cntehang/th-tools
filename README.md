# th-tools

该项目主要是同步携程城市数据到redis。

## 使用方法

Mac系统，在公司网络(连接VPN)中执行。

```bash
# clone 代码
git clone https://github.com/cntehang/th-tools
# 执行同步工具
./th-tools
```
